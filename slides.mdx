<Container >

- `mjml`: Root of the document (like html tag)
- `mj-body`: Content container, you can set bg color, width etc.
- `mj-section`: Like grid row
- `mj-column`: Like grid column

</Container>

<Container>

```
<mjml>
  <mj-body>
    <mj-section>
      <mj-column>
        <!-- First column content -->
      </mj-column>
      <mj-column>
        <!-- Second column content -->
      </mj-column>
    </mj-section>
  </mj-body>
</mjml>
```
